Analizar el siguiente código señalando declaraciones y aplicaciones de reglas, y su efecto.

p#normal {                                 selector
font-family: arial,helvetica;
font-size: 11px;                           declaración             
font-weight: bold;
}

*#destacado {                               selector
border-style: solid;
border-color: blue;                         declaración
border-width: 2px;
}

#distinto {                                 selector
background-color: #9EC7EB;                 declaración
color: red;
}

<p id="normal">Este es un párrafo</p>                             utiliza el estilo de p#normal(letra arial,helvetica negrita tamaño 11 )          

<p id="destacado">Este es otro párrafo</p>                            utiliza el estilo de *#destacado(aplica un borde azul alrededor del parrafo)

<table id="destacado"><tr><td>Esta es una tabla</td></tr></table>    utiliza el estilo de *#destacado(aplica un borde azul alrededor de la tabla)

<p id="distinto">Este es el último párrafo</p>                       utiliza el estilo de #distinto(cin letra roja y color de fondo #9EC7EB)
